Linux Terminal Player

because you don't need to waste 1GB of RAM to listen to music

A lightweight Linux music player with YouTube search, local library support, and background playback. Runs in your terminal and focuses on what matters.

Install now Read README

Features

  • YouTube search and playback via yt-dlp + mpv
  • Local music library folders (Artists and Albums support)
  • Queue and playlists
  • Background playback with daemon
  • Terminal UI with fast keyboard shortcuts

Install

# 1) Dependencies
# Arch
sudo pacman -S mpv yt-dlp

# Ubuntu/Debian
sudo apt install mpv yt-dlp

# Fedora
sudo dnf install mpv yt-dlp

# 2) Build and run
cargo build --release
./target/release/mutui

Tip: mutui starts mutuid automatically when needed.
